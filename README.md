# Employee Management System

A modern and responsive Employee Management System built using React.js and Tailwind CSS.

---

## 🚀 Features

- Admin Dashboard
- Employee Dashboard
- Task Management
- Search Employees
- Responsive Design

---

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- Lucide React (for icons)
- React Router DOM
- Context API

---

## 📦 Libraries Used

### Install Dependencies

```bash
npm install
```

### Install Lucide React

```bash
npm install lucide-react
```

---

## ▶️ Run Project

```bash
npm run dev
```
